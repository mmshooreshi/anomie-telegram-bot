## Application

/start --> /text2link --> {send your text message} --> {make sure your text includes the "secure" word if you want your message to be protected} --> the bot will send you a photo which is created from your text to prevent the text from copying --> the bot will send you a link which is dedicated to your text --> Use this link anywhere you wanted to send protected messages which cannot be read by nobody other than receiver for a limited time.

## Commands

/start 
/merge (To merge multiple texts into one protected message)
/done (To finalize the merging process and receive the URL of the protected message)
/text2link (To create protected message from your single message)
/links (Observe your previous protected messages)

type "secure" in your messages sent to bot, then bot will convert your text message into a photo which has limited time for the observor and can not get copied or forwarded or saved.

## Running the code Locally

Make sure you have [Ruby](https://www.ruby-lang.org), [Bundler](http://bundler.io) and the [Heroku Toolbelt](https://toolbelt.heroku.com/) installed.

```sh
git clone git@github.com:heroku/ruby-sample.git # or clone your own fork
cd ruby-sample
bundle
heroku local
```

Your app should now be running on [localhost:5000](http://localhost:5000/).
